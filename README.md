# FullStack приложение на FastAPI

Проект разработан в соответствии с техническим заданием:  
*"Проектирование, разработка и деплой FullStack приложения на FastAPI"*.

## Соответствие требованиям

- **Технологический стек**: Python 3.11, FastAPI 0.104+, PostgreSQL 15, SQLAlchemy 2.0+, Pydantic v2, Render.com  
- **Предметная область**: Блог (сущности: Авторы и Посты)  
- **Архитектура БД**: Две связанные таблицы (`authors`, `posts`) с отношением один-ко-многим через внешний ключ и ORM-связи в SQLAlchemy  
- **REST API**: Реализованы CRUD-операции для авторов и постов с префиксом `/api/v1`  
- **Документация API**: Автоматическая OpenAPI-документация доступна по маршруту `/docs`  
- **Фронтенд**: Базовая HTML-страница с навигацией на документацию API  
- **Безопасность**: Учетные данные базы данных и секретные ключи передаются через переменные окружения (`DATABASE_URL`, `SECRET_KEY`)  
- **Деплой**: Приложение и база данных развернуты на Render.com  
- **Структура проекта**: Соответствует рекомендованной — разделение на `app/`, `models.py`, `schemas.py`, `routers/`, `templates/`
